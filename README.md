# OpenGL Project
This project is supposed to be a resource to use for other OpenGL-based projects.

![Basic OpenGL Triangle](./res/images/basic-opengl-triangle.png)

## Quickstart
```bash
$ make release
$ bin/release/opengl-proj
```
